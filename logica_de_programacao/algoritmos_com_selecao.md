## Algoritmos com seleção

#### Seleção simples
~~~
if (<condition = true>) <then> {
    <instructions>
}
~~~

#### Seleção composta ou dupla
~~~
if (<condition = true>) <then> {
    <instructions>
} else <condition = false> {
    <instructions>
}
~~~

#### Seleção concatenada ou aninhada
~~~
if (<condition = true>) <then> {
    if (<condition = true>) <then> {
        <instructions>
    }
}
~~~

#### Seleção múltipla
`switch case`
