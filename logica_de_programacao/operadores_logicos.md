## Operadores Lógicos

Os operadores lógicos são utilizados em comparações que retornam um valor **verdadeiro** ou **falso**.

| Símbolo | Comparação |
| :------------: | :------------: |
| == | igual a |
| =! | diferente de |
| > | maior que |
| < | menor que |
| <= | menor ou igual a |
| >= | maior ou igual a |
